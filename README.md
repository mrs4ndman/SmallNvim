- ## Installation / Instalación:
```bash
git clone git@github.com:mrs4ndman/SmallNvim.git
mv ~/.config/nvim/init.lua ~/.config/nvim/init.lua.bak
mv init.lua ~/.config/nvim/init.lua
```
